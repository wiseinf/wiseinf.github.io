Cluster Optimizer v1.3.0版已发布

主要描述集群优化平台Cluster Optimizer1.3.0版本的新特性。
2024-07-10

集群优化平台Cluster Optimizer版本1.3.0已发布。

主要特性包括：

  1. 商业版 自动化动作：支持自动化的调整数据存储卷（Persistent Volume）的容量大小，目前仅限于Deployment。

Cluster Optimizer v1.2.0版已发布

主要描述集群优化平台Cluster Optimizer1.2.0版本的新特性。
2024-05-20

集群优化平台Cluster Optimizer版本1.2.0已发布。

主要特性包括：

  1. 认证和授权：支持用户认证和权限，为用户提供了管理员和用户两种角色，管理员具备所有操作权限，用户仅有查看权限。
  2. 集群访问账户：支持管理用户的集群访问账户，能够为每个用户设置集群级别的集群访问账户，包括该用户的云服务商账号，以及集群的Kubeconfig等配置。用户可以通过此身份来操作相应的集群。同时，若集群支持自动更新Kubeconfig，则用户的集群访问账户在使用时也可以定时更新。
  3. 自动化动作：支持应用级的资源推荐、未设置资源额度和内存溢出，以及删除未使用的持久卷等自动化动作，支持标准的工作负载Deployment、StatefulSet、DaemonSet和CronJob。不支持Job，其资源额度（Resource Quota）不可修改。
  4. 任务管理：支持创建、运行、重运行、取消、查看任务日志、删除任务等日常任务管理操作。

快速部署和试用Cluster Optimizer，请参见：免费试用。

Cluster Optimizer v1.1.0版已发布

主要描述集群优化平台Cluster Optimizer1.1.0版本的新特性。
2024-04-09

集群优化平台Cluster Optimizer版本1.1.0已发布。

主要特性包括：

  1. AWS Cloud EKS集成：支持导入AWS EKS集群，查看针对集群的优化建议。
  2. GPU节点：支持查看GPU利用率低的GPU节点。
  3. 快速资源关联：支持通过资源的标签快速将资源和相应的部门、组、用户关联起来，并能够展示到Web界面。
  4. 邮件通知：针对具体的优化策略，用户可以订阅邮件通知，方便通过邮件了解到相应的优化建议。
  5. 集群状态：能够展示导入集群中的Kubernetes状态和Metrics服务的状态。对于异常状态，能够快速发现到异常原因。
  6. 支持自动创建数据库Schema：可通过数据库连接配置项中的参数createDatabaseIfNotExist=true来指明在数据库不存在是是否自动创建数据库。当指定的用户名和密码具备创建数据库权限时，系统将自动创建数据库。

快速部署和试用Cluster Optimizer，请参见：免费试用。
