Ruyi：云原生集群扩缩容引擎

Ruyi（即如意，意味着集群能够随心所欲的进行扩缩容）是云智优本公司研发的一款集群自动扩缩容软件。它能够监控调度在节点组（或节点池）上的Pod Request值，并基于此值对节点组进行扩缩容。

它能够广泛的应用于在线服务、批量任务和大数据等场景中，帮助企业在性能和成本间取得平衡。
