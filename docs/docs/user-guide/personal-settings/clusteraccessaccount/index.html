Cluster Access Account

Describes operations related to managing personal cluster access accounts.
Tags:
  • Cluster Optimizer
  • Cluster Access Account
Categories:
  • Cluster Access Account

Describes operations related to managing personal cluster access accounts.

Preconditions

  • Available only in commercial version.

View cluster access account

Click on the user avatar in the top left navigation bar, as shown below. Click Personal Settings on the expanded menu to enter the personal settings page.

On the personal settings page, click the Cluster Access Account tab to view and manage the cluster access account, as shown in the figure below.

Please see the table below for specific field names and descriptions.

Field name Description
Cluster ID User name, required
Cloud service provider access account User last name, required
Kubeconfig User email address, required
Operation Display user operations

Enter the corresponding information according to the field name and click the OK button to modify the personal information.

Add cluster access account

Click the Add button at the top of the cluster access account management page, and the Add Cluster Access Account dialog box will pop up, as shown below.

Please see the table below for specific field names and descriptions.

Field name Description
Cluster ID Accessed cluster ID, required
Access Key Access key to access the cloud service provider, required
Access Key Secret Access key to access the cloud service provider, required
Kubeconfig Credentials for accessing Kubernetes, required

Enter the corresponding information according to the field name and click the OK button to add the cluster access account.

Note: If you have not added a cluster, the drop-down box of the cluster ID field is empty, the system will report an error, and the cluster access account cannot be added. You can create a cluster access account after [Importing the cluster](/docs/user-guide/import-cluster/].

Update cluster access account

Click the Cluster Access Account button in the list of cluster access account management pages, and the Edit Cluster Access Account dialog box will pop up, as shown below.

The specific field names and descriptions are the same as [Add Cluster Access Account] (#Add Cluster Access Account) and will not be repeated.

Delete cluster access account

Click the Delete button on the cluster access account management page to delete the cluster access account.

Before deleting the cluster access account, a deletion confirmation dialog box will pop up. After confirmation, the deletion operation will be performed.
