Personal Account Settings

Describe operations related to personal account settings.

Describe operations related to personal account settings.

Preconditions

  • Available only in commercial version.

Update personal information

Click on the user avatar in the top left navigation bar, as shown below. Click Personal Settings on the expanded menu to enter the personal settings page.

On the personal settings page, click the Personal Account tab to view and update personal information, as shown in the figure below.

Please see the table below for specific field names and descriptions.

Field name Description
Name User name, required
Last name User last name, optional
First Name Username, optional
Email User email, required
Password User login password, required
Confirm password Confirm user login password, required
Mobile phone number User mobile phone number, required
User role Currently, the system provides two roles: administrator and user. The default is user, which is required
Regional Settings Currently supports Chinese and English locales.

Enter the corresponding information according to the field name and click the OK button to modify the personal information.
