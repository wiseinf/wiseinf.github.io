Notification Channel

Describes how to set up notification channels.

Describes how to manage notification channels, including viewing, adding, updating, or deleting notification channels.

View channels

Enter the notification channel management page from Settings --> Notification Channels, as shown below.

The notification channel management page will display notification channels in the form of a list. It mainly includes 5 columns:

  • Name: Notification channel name.
  • Channel type: Notification channel type.
  • Connection parameters: Displays the connection parameters of this channel type.
  • Other parameters: display other parameters
  • Operation: Operation bar, including editing, deletion and other operations.

Add channel

Click the Add button at the top of the notification channel management page, select the notification channel type, and fill in the notification channel information, as shown below.

Please see the table below for specific field names and descriptions.

Field name Description
Name Notification channel name
Channel type Notification channel type, currently only supports Email
Host Email SMTP hostname, for example smtp.wiseinf.com
Port Email SMTP port, such as 25, 465
Enable SSL Whether to enable SSL, the default is not enabled
Email Email address, such as admin@wiseinf.com
Sender name Sender name, e.g. Wiseinf
Password Email SMTP Password
Parameter list User-defined parameter list in the form of key-value pairs. This parameter list will be passed to the template of the notification channel and can be referenced in the template through {{.param.parameter name}}.

Enter the corresponding information according to the field name and click the OK button to add the notification channel.

For specific values ​​of the above fields, you can consult your email service provider or email system administrator.

Update channel

To update notification channels, click the Edit button in the list of notification channel management pages to update the notification channel information, as shown below.

The specific field names and descriptions are the same as [Add Channel] (#Add Channel) and will not be repeated.

Delete channel

Click the Delete button in the list of notification channel management pages to delete the notification channel, as shown below.

Before deleting the notification channel, a deletion confirmation dialog box will pop up, and the deletion operation will be performed after the user confirms.

Note: When the notification channel is used by notification rules, it will prompt that the deletion was unsuccessful. You should first delete the notification rules that use the notification channel, and then delete the notification channel.
