1 - Adjusting Node Group

Adjust node group.

Describes the operation of adjusting a node group.

Alibaba Cloud

Prerequisites

  • Have permission to edit node groups (node pool).

Operation steps

  1. Log in to the Container Service Management Console and select the cluster in the left navigation bar.
  2. On the cluster list page, click the target cluster name, and then select Node Management > Node Pool in the left navigation bar.
  3. On the node pool list page, click Edit in the Action column of the row where the target node pool is located.
  4. On the Edit Node Pool page, edit the configuration items of the node pool. Node pool optimization mainly focuses on instance specifications and automatic scaling configuration items.
    1. According to the recommended instance rules in the Adjust Node Pool dialog box, select the corresponding instance specification in Instance Specifications.
    2. Make adjustments in the Auto Scaling configuration item based on the recommended automatic scaling, minimum number of nodes, and maximum number of nodes in the Adjust Node Pool dialog box. If cluster automatic scaling is not enabled, please refer to Node automatic scaling to configure.

For more details, please refer to: Edit Node Pool
