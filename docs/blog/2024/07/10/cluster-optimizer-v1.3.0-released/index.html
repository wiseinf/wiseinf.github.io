Cluster Optimizer v1.3.0 Released

Overview of the new features in Cluster Optimizer v1.3.0.
2024-07-10

The cluster optimization platform Cluster Optimizer v1.3.0 has been released.

Key features include:

  1. Automated Actions: Supports resizing of persistent volumes (PVs), currently limited to Deployment.
  2. Cluster Hibernation: Introduces the cluster hibernation feature, allowing manual or scheduled hibernation of clusters to free up node resources, with the ability to restore both the cluster and its workload states upon waking.

To quickly deploy and try Cluster Optimizer, please refer to: Free Trial.
